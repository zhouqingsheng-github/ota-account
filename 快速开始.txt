GitHub Actions 自动打包 - 快速命令

=== 第一步：初始化Git ===
cd python-tool
git init
git add .
git commit -m "Initial commit: OTA凭证获取工具"

=== 第二步：创建GitHub仓库 ===
1. 访问 https://github.com/new
2. 仓库名: ota-credential-tool
3. 选择 Public 或 Private
4. 点击 Create repository

=== 第三步：推送代码 ===
git remote add origin https://github.com/你的用户名/ota-credential-tool.git
git branch -M main
git push -u origin main

=== 第四步：触发打包 ===
git tag v1.0.0
git push origin v1.0.0

=== 第五步：下载文件 ===
1. 进入 GitHub 仓库
2. 点击 Actions 标签
3. 等待打包完成（5-10分钟）
4. 下载 Artifacts 中的文件

完成！你将得到三个平台的可执行文件。

详细教程请查看: GITHUB_ACTIONS_GUIDE.md
